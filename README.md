# Create a Responsive Login Page in HTML and CSS | Responsive Login Form in HTML & CSS

Um projeto simples de **página de login responsiva** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar uma **login page moderna e responsiva**, adaptável para diferentes tamanhos de tela.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Página de login responsiva
- Layout adaptável (mobile e desktop)
- Design simples e moderno

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
